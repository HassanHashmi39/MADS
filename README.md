# MADS Website

Experience modern web development at its best. Built with precision and a focus on premium design.

## 🚀 Getting Started

Follow these steps to set up the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/HassanHashmi39/MADS.git
   cd mads-website
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

## 🛠️ Scripts

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Builds the application for production.
- `npm run lint`: Runs ESLint for code quality checks.
- `npm run preview`: Locally previews the production build.

## 📁 Project Structure

- `src/`: Core application logic and components.
- `public/`: Static assets accessible at the root level.
- `assets/`: Optimized images and media.
- `.github/workflows/`: CI/CD automation.

## 🤝 Contribution Guidelines

- Follow existing code patterns and keep components modular.
- Ensure all PRs pass the CI lint and build checks.
- Use clear and descriptive commit messages.

## 🌐 Deployment

The project is configured for automated deployment via Netlify. Pushing to the `main` branch triggers a new production build.
